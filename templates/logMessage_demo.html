Demo: logMessage

Muestra en vivo los mensajes que envía el agente usando el Client Tool logMessage.
Sesión
—
⏳ Esperando a que inicie la llamada…

Cómo probar:

  1. Reemplaza agent_4001kc571fptfk0vj47krstyn85v en el HTML por tu Agent ID.
  2. En el dashboard del agente, crea el Client Tool logMessage con parámetros:
    • level (string, requerido; ej. "info", "warn", "error")
    • message (string, requerido)
    • data (json/string, opcional)
  3. Pega el system prompt incluido en el archivo system_prompt_logMessage.txt.
  4. Abre la página, pulsa “Start call” y acepta micrófono.
